### Features

### Installation

No installation required.

This project is distributed as a **standalone Windows executable (.exe)**.  
Python and dependencies are already included.

### Usage

1. Launch the `.exe` file.
2. Enter a short password (3 characters).
3. Click **Simulate**.
4. Observe how fast weak passwords can be cracked.
